# LED Blink Project

## Project Title
Arduino LED Blink Project

## Hardware Required
- Arduino UNO
- LED
- 220Ω Resistor
- Potentiometer
- Breadboard
- Jumper Wires

## Circuit Diagram Description
The LED is connected to digital pin 13 through a resistor. The potentiometer is connected to analog pin A0. The Arduino supplies 5V and GND to the circuit.

## How to Upload Code
1. Connect the Arduino to the computer.
2. Open Arduino IDE.
3. Open led_blink.ino.
4. Select Arduino UNO as the board.
5. Select the correct COM port.
6. Click Upload.

## Expected Output
The LED blinks continuously. The blink speed changes according to the potentiometer position, and the Serial Monitor displays the blink count.

## Troubleshooting Tips
1. Check the USB cable connection.
2. Verify the correct COM port is selected.
3. Check all wiring connections and component polarity.
